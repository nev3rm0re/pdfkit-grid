pdfkitgrid.border(startX, startY, width, height, options) ⇒

Creates border around specified area

You can specify which borders should be drawn, using options.border property. border can be either boolean, or a string indicating "directions" to draw:

N - north or top S - south or bottom E - east or right W - west or left

border of "NSEW" is identical to true - all borders will be drawn

Kind: static method of pdfkitgrid
Returns: PDFDocument

Param Type Default
startX number
startY number
width number
height number
options Object {}
